# Todo App

Next.js + TypeScript + Tailwind CSS で作成したシンプルな Todo アプリです。

## 機能

- Todo の一覧表示
- Todo の追加
- Todo のメモ追加・表示・編集
- Todo の期限日追加・表示・編集
- Enter キーでの Todo 追加
- Todo の完了切り替え
- Todo タイトルの編集
- Enter キーでの編集保存
- Esc キーでの編集キャンセル
- 空文字の追加・保存防止
- Todo の削除
- 「すべて」「今日」「予定」「未完了」「完了済み」の表示フィルタ

## 開発

```bash
npm.cmd run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開くと確認できます。

## テスト

```bash
npm.cmd test
```
