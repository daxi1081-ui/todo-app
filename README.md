# Todo App

Next.js + TypeScript + Tailwind CSS で作成したシンプルな Todo アプリです。

## 機能

- Todo の一覧表示
- Todo の追加
- Todo の完了切り替え
- Todo のタイトル編集
- Todo の削除
- 「すべて」「未完了」「完了済み」の表示フィルタ

## 開発

```bash
npm.cmd run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開くと確認できます。

## テスト

```bash
npm.cmd test
```
